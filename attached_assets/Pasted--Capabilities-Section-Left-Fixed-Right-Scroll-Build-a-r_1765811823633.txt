(Capabilities Section — Left Fixed, Right Scroll)

Build a responsive Capabilities section for the Gray Solutions website with a 2-column layout:

1) Layout + Behavior

Desktop layout: Left column fixed/sticky, Right column scrollable.

Left column stays visible while the user scrolls through the right column content.

Right column contains a stacked accordion list (collapsible items).

Only one accordion item open at a time (opening one closes the others).

The right column must be internally scrollable (not page scroll) with a visible but minimal scrollbar (thin).

Section height should feel like a “panel” (e.g., min-height: 80vh on desktop).

2) Styling (Minimal, Premium)

Clean white/light background (match overall site theme).

Typography: modern sans (use system font stack if no font configured).

Spacing: generous padding, aligned grid, no clutter.

Accordion cards: full-width rows with rounded corners, subtle border, hover state, and smooth expand/collapse animation.

Right side items should look like large pill/tiles:

Closed state: title left, plus icon right.

Open state: title left, minus icon right + body content revealed below.

Body content area: short paragraph + 3 bullets + “Explore →” link.

Make it accessible: buttons with aria-expanded, keyboard navigation, proper focus rings.

3) Responsive Rules

Mobile (<768px): switch to single column.

Left column becomes top header block (not sticky).

Accordion list below, normal page scroll (no internal scroll).

Tablet: sticky can remain, but reduce padding and font sizes slightly.
